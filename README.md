# Proyecto_Logica_Katas_Python
Repositorio creado con el fin de realizar la entrega del proyecto de lógica de Katas de Python del bootcamp de thePowerEducation

# Descripción del proyecto

Este proyecto consiste en resolver una serie de ejercicios (katas) para poner en práctica los conocimientos adquiridos durante el módulo de Python.

# Contenidos trabajados

A lo largo del proyecto se han utilizado:

* Tipos de datos básicos
* Condicionales (if, elif, else)
* Bucles (for, while)
* Funciones
* map, filter y reduce
* Manejo de excepciones (try, except)
* Clases (programación orientada a objetos)

# Pasos seguidos

1. Leer y entender cada enunciado
2. Pensar la lógica antes de resolver los enunciados
3. Resolver cada ejercicio usando funciones
4. Probar el código con distintos ejemplos
5. Corregir errores y mejorar el código para hacer versiones más simplificadas

# Estructura del proyecto

* `proyecto_logica_katas_resuelto.py`: archivo con los ejercicios resueltos
* `README.md`: explicación del proyecto

# Conclusión

Este proyecto me ha ayudado a mejorar la lógica de programación y a entender mejor cómo resolver problemas con Python. También me ha servido para practicar diferentes formas de llegar a un mismo resultado.
